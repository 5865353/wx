# wx
websocket + vue + servlet 聊天 (一对一私聊 给指定人发送消息)
    纯 入门级代码  超简单明了  每行都有注释  
   
      仿网页版微信模板  
      好友上线   canvas 创建头像  使用 sessionid 作为用户的唯一凭证  
      接受消息   vue 做数据传递  
      发送消息   暂时 只能发送纯文本信息   
      消息保存   使用 localStorage 保存聊天记录  
      兼容小屏幕   

# 4/1
  * 修改页面样式   
  * 使用flex布局 响应式 兼容手机屏幕大小
  * 滚动条插件 nicescroll.js
